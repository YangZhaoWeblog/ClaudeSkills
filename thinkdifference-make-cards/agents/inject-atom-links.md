# 原子笔记链接注入 Agent

你的任务：为一份 Anki 卡片文件注入原子笔记链接，使复习时可一跳到达知识图谱节点。

## 执行流程

1. 用 Glob 获取 `1.原子笔记/*.md` 文件名列表
2. 读取每个文件的 frontmatter `aliases` 字段，构建匹配表：`{文件名: [aliases]}`
3. 读取目标 Anki 卡片文件
4. 对每张卡片的 A: 内容，识别其核心概念，按优先级匹配：
   文件名精确匹配 > alias 匹配 > 不插入
5. 在 A: 最末行（`---` 分隔符之前）追加 `→ [[匹配到的文件名]]`
6. 用 Edit 工具回写文件

## 硬约束

- 链接只放 A: 最末行，**禁止**插入答案正文中间
- 匹配不上则不插入——**宁缺毋错，禁止杜撰不存在的文件名**
- 若 `1.原子笔记/` 为空或不存在，直接返回，不报错

## 示例

✅ 正确：
```
Q: 在 RSA 加密中，已知公钥和密文，为什么无法反推明文？
A: 解密需要私钥 d → 需要 φ(n) → 需要 p,q → 只能分解 n → 计算不可行
→ [[大整数分解困难性]]

---
```

❌ 链接插在答案中间：
```
A: 解密需要 [[大整数分解困难性]] 这个数学难题来保障...
```

❌ 杜撰文件名：
```
→ [[大数分解难题]]   ← 实际文件名是"大整数分解困难性.md"
```
