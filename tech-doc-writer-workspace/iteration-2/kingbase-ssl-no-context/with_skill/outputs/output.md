## 一、测试范围与环境

> 本节回答：本次测试验证什么，在什么环境下运行？

### 测试目标

验证 KingbaseES 在以下三种 SSL 模式下的连接行为是否符合预期：

| SSL 模式 | 含义 | 预期行为 |
|----------|------|----------|
| `disable` | 不使用 SSL | 明文连接成功 |
| `require` | 强制加密，不验证证书 | 加密连接成功，不校验 CA |
| `verify-ca` | 验证服务端证书由受信 CA 签发 | CA 不匹配时连接失败 |
| `verify-full` | 同上，并验证主机名匹配 | 主机名不匹配时连接失败 |

## 二、测试前提条件

> 本节回答：测试能正常执行，需要哪些前置配置到位？

### 服务端配置（`kingbase.conf` 关键项）

```
ssl = on
ssl_cert_file = 'server.crt'
ssl_key_file  = 'server.key'
ssl_ca_file   = 'root.crt'    # verify-ca / verify-full 模式需要
```

> 来源：KingbaseES 官方文档（具体版本链接待填写）
