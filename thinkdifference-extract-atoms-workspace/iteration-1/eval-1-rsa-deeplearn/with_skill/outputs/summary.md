## 原子知识点提取清单

来源：[[RSA加密-deeplearn]]

### 核心原子（已创建）
- [[单向函数]] — 正着算一瞬间，反着算难到绝望——打碎盘子容易，拼回去几乎不可能
- [[大整数分解困难性]] — 两个大素数相乘是毫秒级操作，把乘积拆回两个素数是世纪级难题
- [[欧拉函数]] — 小于 n 且和 n "没有公因数关系"的数有多少个——知道 n 的素因子就能秒算，不知道就得硬数
- [[欧拉定理]] — a^phi(n) mod n = 1——自乘 phi(n) 次后取模必然回到 1，像钟表转整圈回到原点
- [[模逆元]] — 找一个数 d 使得 e*d mod n = 1——模运算世界里的"除法"
- [[公钥密码学]] — 加密和解密用不同的钥匙——公钥像信箱投递口人人可用，私钥像信箱钥匙只有主人持有

### 边缘原子（已创建）
- [[模运算]] — 数字绕一个圈，绕完回到起点——信息在"绕圈"中不可逆地丢失
- [[互质]] — 两个数除了 1 之外没有公因数——判定方法是 gcd(a, b) = 1
